# Contexto de la Aplicación
## Introducción
La aplicación se divide en tres secciones principales: Backend, Frontend y DevOps.
### Backend
El backend se encarga de la lógica de la aplicación y se encuentra en el directorio ./API.
### Frontend
El frontend se encarga de la interfaz de usuario y se encuentra en el directorio ./Android.
### DevOps
El DevOps se encarga de la implementación y mantenimiento de la aplicación.
